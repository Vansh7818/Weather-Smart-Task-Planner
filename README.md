# Weather-Smart Task Planner

An AI-powered tool that intelligently schedules your indoor and outdoor activities based on real-time weather forecasts with transparent reasoning.

## Features
- Schedules outdoor tasks at the most comfortable time
- Provides AI reasoning for every task
- Handles indoor tasks automatically
- Visual temperature trend graph
- Detects severe weather conditions

## Tech Stack
- Python 3.12
- Streamlit
- OpenWeatherMap API
- Pandas & Matplotlib for visualization
